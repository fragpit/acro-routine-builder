# Trick rules and constraints

Extracted from
[sporting_code_aerobatics_2025.md](sporting_code_aerobatics_2025.md).

## 1. Position restrictions

### 1.1 "Must be first" restriction

Flat Stall to Infinity Tumbling MUST be the first manoeuver
in a run.

### 1.2 "Cannot be one of the last two manoeuvres"

The following manoeuvres cannot be one of the last two
manoeuvres in a run:

- Tumbling
- Anti-Rhythmic SAT
- Infinity Tumbling
- Mac Twist to Tumbling
- Rhythmic SAT
- Esfera
- X-Chopper to Tumbling
- Super Stall to Infinity Tumbling
- Flat Stall to Infinity Tumbling
- MacFly, MistyFly, HeliFly, SatFly
- Misty to Tumbling

A flipped manoeuvre cannot be one of the last 2 manoeuvres.

## 2. Forbidden connections (sequence restrictions)

Each manoeuvre may have forbidden connections - the manoeuvre
listed in the right column cannot directly follow the one
on the left.

"SAT-based combo manoeuvres" means combo tricks that start
from SAT (i.e. begin with a SAT entry): SAT to Helicopter,
SatFly. Combos that end with SAT (Misty to SAT, X-Chopper to
SAT, Mactwist to SAT, Helicopter to SAT) start with a
different entry, so they are allowed after a plain SAT.

| Manoeuvre                         | Forbidden connection to           |
| --------------------------------- | --------------------------------- |
| SAT                               | SAT-based combo manoeuvres        |
| Misty to SAT                      | SAT                               |
| Helicopter                        | any helicopter connection         |
| Misty to Helicopter               | Helicopter                        |
| Twister                           | Helicopter                        |
| Tumbling                          | Infinite and Anti-Rhythmic        |
| SAT to Helicopter                 | Helicopter                        |
| Cowboy                            | Helicopter                        |
| SuperCowboy                       | Helicopter                        |
| Corkscrew                         | Helicopter                        |
| Joker                             | Helicopter                        |
| Mac Twist to Tumbling             | Infinite Tumbling, Anti-Rhythmic  |
| Rhythmic SAT                      | Tumbling, Infinite, Anti-Rhythmic |
| Infinity Tumbling                 | Tumbling, Anti-Rhythmic SAT       |
| X-Chopper to Tumbling             | Infinite, Anti-Rhythmic           |
| Super Stall to Infinity           | Tumbling, Anti-Rhythmic SAT       |
| Flat Stall to Infinity            | Tumbling, Anti-Rhythmic SAT       |
| MacFly, MistyFly, HeliFly, SatFly | Tumbling, Anti-Rhythmic SAT       |

## 3. Per-run limits

### 3.1 High coefficient manoeuvres limit

Maximum 2 manoeuvres with coefficient >= 1.95 per run.
Extra manoeuvres will not be scored and their coefficients
not taken into account for the average coefficient (technicity).

### 3.2 One-per-run: stall-to-infinite family

Only one of the following manoeuvres can be performed during
the same run:

- Super Stall to Infinite Tumbling
- Flat Stall to Infinite Tumbling
- Tail Slide to Infinite Tumbling
- MacFly (Mactwist to Tailslide to Infinity Tumbling)
- MistyFly (Misty to Tailslide to Infinity Tumbling)
- HeliFly (Helico to Tailslide to Infinity Tumbling)
- SatFly (SAT to Tailslide to Infinity Tumbling)

### 3.3 Tumbling/infinity/rhythmic limit

Only 2 tumbling, infinity and rhythmic related manoeuvres
can be performed during the same run.

### 3.4 Mutual exclusion: X-Chopper/Misty to Tumbling vs stall-to-infinite

X-Chopper to Tumbling and Misty to Tumbling cannot be
performed in the same run as a stall-to-infinite type
manoeuvre (see 3.2).

### 3.5 Bonus limits per run

- Maximum 5 manoeuvres can be performed twisted per run
- Maximum 3 manoeuvres can be performed reversed per run
- Maximum 2 manoeuvres can be performed flipped per run

Extra manoeuvres beyond these limits will not be scored and
their coefficients not taken into account for the average
coefficient.

Flipped manoeuvres must be announced before the run.

> **Competitions above ground:** flipped manoeuvres
> are forbidden.

### 3.6 No-side manoeuvres (once per run)

MacFly, MistyFly, HeliFly, SatFly - each doesn't have
a side and can be performed only once.

## 4. Per-competition repetition rules

### 4.1 General rule

Each manoeuvre may only be performed once in each direction
during the same competition without deduction. A twisted or
flipped manoeuvre is considered the same manoeuvre as
an untwisted one. A reverse manoeuvre is considered
a different manoeuvre altogether.

Example: you can do a Joker to the left, followed by
a twisted Joker to the right, followed by a twisted Joker
reverse to the left without penalty. But if you do a twisted
Joker to the left and a Joker to the left you will get
one malus.

### 4.2 Repetition penalty

The penalty for repetition is typically a reduction of 13%
of the choreography mark for each repetition.

### 4.3 Exceptions (repetition allowed)

The following manoeuvres can be performed more than once
without penalty within the same competition:

- Tail Slide
- Wingover
- Full Stall
- Super Stall
- Pitch Pendulum
- Bro Hook

However, each manoeuvre can be performed only once twisted
or flipped within the same run. If repeated twisted or flipped
within the same run, the extra bonus will not be rewarded.

### 4.4 AWT-specific: Misty to Misty

In the AWT, Misty to Misty does not exist and cannot be used
as a manoeuvre. If a pilot flies 2 Misty Flips in a row it
will be counted as 2 separate Misty Flips.

## 5. Program start announcement

Before starting the program, every pilot must make one
"big ear" to notify the judges of the start of the run.
Omission leads to a pre-warning, next omission leads to
an official warning.
