# News

## v0.9.12

- Bonus-limit warnings now highlight only the extra twisted, reversed or flipped
  tricks that will not be scored.

## Flipped bonuses now block other bonuses

Flipped variants are now treated as exclusive bonus choices. When you select a
Flip or Double Flip bonus, every other bonus on that trick is disabled, and the
same rule is applied to default bonuses when a trick is inserted.

This keeps the builder aligned with the intended scoring model and prevents
accidental combinations such as Flip plus Twisted, Reverse, Hardcore Entry or
Cab Slide on the same trick.
